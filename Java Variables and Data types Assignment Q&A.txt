JAVA VARIABLES AND DATA TYPES

1.What is statically typed and dynamically typed programming language?

Statically types language need to specify data type before assigning a value to variable
Dynamically typed languages need not to specify data type before assigning a value to a variable.

2.What is the variable in java?

It is an assigned memory location where a value of particular data type is stored.

3.How to assign a value to a variable?

We use ‘=’ to assign a value to a variable.

4.What are primitive data types in java?

These predefined data types are already present in java.
Example-byte, short, int, long, char, boolean, float, double

5.What are identifiers in java?

It is a name given to a package, class, variable or methods.
All identifiers must have different names.

Arithmetic: +, -, *, /, %, ++, --
Assignment: =, +=, -=, *=, /=, %=
Relational: ==, !=, >, <, >=, <=
Logical: &&, ||, !
Bitwise: &, |, ^, ~, <<, >>, >>>
