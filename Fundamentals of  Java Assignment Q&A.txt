FUNDAMENTALS OF JAVA


1.What is a programming language?

It is a type of high level language which is compiled to generate native machine code which in turn gets executed according to the set of instructions present in the program.

2.Why do we need a programming language?

A programming language is a more user friendly way to establish a defined communication protocol between a user and a computer.

3.What are the features of java?

Java is an object oriented programming language?
It follows the principle of WORA(Write once run anywhere)
It is portable in nature
It is platform independent

4.What is an object?

It is an instance of class.

5.What is a class?

It is a blueprint in which specific data members and methods are defined.

6.Explain about the main() method in Java?

It is the starting point from which the JVM starts the execution of java program
